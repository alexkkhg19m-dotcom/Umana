🔑 GitHub Token (publicación automática)

Para que al publicar un artículo se suba automáticamente a GitHub (y Google lo indexe), añade un Personal Access Token. Se guarda solo en este navegador y solo se usa para subir archivos al repositorio.

Crea un PAT en github.com/settings/tokens con permisos repo (o contents:write para Fine-Grained).

📋 Pasos para configurar (una sola vez):
  1. Inicia sesión en GitHub con la cuenta Umana-Salud-Mental
  2. Ve a Settings → Developer Settings → Fine-grained tokens
  3. Clic en Generate new token | Nombre: Umana Admin
  4. Repository access → Only selected → umana-salud-mental.github.io
  5. Permissions → Contents: Read and write
  6. Copia el token (github_pat_...) y pégalo arriba
  7. Deja el campo Rama en main
  8. Clic en 💾 Guardar Token

🔍 SEO & Google

Con el token configurado, publicar un artículo lo sube automáticamente a GitHub. Cada artículo obtiene su propia URL estática indexable por Google.

La URL pública será: https://umana-salud-mental.github.io/posts/post-[id].html

📋 ¿Qué hace "Publicar Sitemap en GitHub"?

Sube un archivo sitemap.xml a tu repositorio. Google lee ese archivo automáticamente y encuentra todas las URLs de tus artículos.

Haz clic en este botón cada vez que publiques un artículo nuevo, para que Google lo encuentre más rápido (en 1–7 días).
