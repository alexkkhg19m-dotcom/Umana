Admin de Umana Blog

Para publicar artículos necesitas un token de acceso de GitHub. Se guarda solo en tu navegador y nunca se comparte con nadie.

¿Cómo crear tu token?
  1. Inicia sesión en GitHub con la cuenta Umana-Salud-Mental
  2. Ve a Settings → Developer Settings → Fine-grained tokens
  3. Clic en "Generate new token"
  4. Nombre: Umana Admin | Expiración: No expiration
  5. Repository access: Only selected → umana-salud-mental.github.io
  6. Permissions → Contents: Read and write
  7. Clic en "Generate token" y copia el token (github_pat_...)

Se guarda solo en este navegador (localStorage). No se envía a ningún servidor.
